Gemini API Key Configuration

EduMind operates in two modes:

  • Local Simulation Mode (Default): Standard rule engine structures schedules and mock-evaluates quizzes offline. Ideal for demonstration without external credentials.
  • Live Gemini Mode: Connects to real Gemini models via direct HTTP requests to evaluate your quizzes and create unique, customized roadmaps.

🔑 Your key is stored solely in your browser's session storage and sent directly to Google APIs in-memory. It is never stored on disk or sent to any other service.
